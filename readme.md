## 🎟️ 클룩 할인코드 적용 방법

1. 클룩에서 원하는 상품을 선택  
2. 결제 화면에서 **‘프로모션 코드’ 입력란**에 할인코드 붙여넣기  
3. 할인 금액이 적용되었는지 확인 후 결제  

예시: `KLOOKAPRIL24`
